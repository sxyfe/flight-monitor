Skill · flight-monitor-agent

在 IDE 里
说一句话，出报告。

flight-monitor-agent 是独立开源 Skill：自然语言 → 实时 RollingGo 查价 → 暖色分页 HTML 报告。与本站 在线查价 共用同一套搜索引擎，适合在开发工作流里快速扫东南亚/日本特价、开口程与国庆窗。

安装

先在 rollinggo.store 申请 RollingGo MCP API Key（格式 mcp_...）。

推荐通过 skills CLI 安装：

npx skills add sxyfe/skills@flight-monitor-agent

或从 GitHub 仓库手动安装：

# 仓库
https://github.com/sxyfe/skills

安装后复制 .env.example 为 .env，填入 Key，并运行连通性检查： 

python3 scripts/check_rollinggo.py

工作流

01

解析意图

parse_nl_intent.py 将自然语言转为 SearchIntent，并给出精简/全量两种预估 API 次数。

02

选择模式

精简模式（smart）日常扫价；全量穷举（exhaustive）覆盖全部城市组合。

03

实时查价

run_search.py 调用 RollingGo，禁止读取历史 JSON 缓存代替查价。

04

HTML 报告

generate_flight_report.py 生成自包含暖色报告：五图联动 + 筛选 + 分页表格。

Web 查价 vs Skill

在线查价（nl-search） Skill
使用场景 浏览器，表单/矩阵/NL 三模式 Cursor 对话 + 脚本流水线
结果形态 页内暖色报告 + SSE 进度 本地 output/reports/…/report.html
Key 配置 Web 设置页 / 环境变量 .env 或 ~/.cursor/mcp.json
穷举模式 精简 / 全量，可中途停止 精简 / 全量，高成本需确认

域知识速查

  • 开口程：去程 A→B、返程 C→D；A/D 在出发地集合内，B/C 任意
  • 国庆 → 10-01~10-07；国庆前后 → 9-28~10-10
  • 开口程价为两段 ONE_WAY 相加，bookable: false
  • 禁止：用历史 exhaustive_results 代替实时查价；跳过模式选择与高额确认
