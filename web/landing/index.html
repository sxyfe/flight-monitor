三步完成一次查价

01

说意图

自然语言或表单：多出发地、多国目的地、停留天数、开口程/往返。

02

选模式

精简模式快速扫热门城市；全量穷举覆盖配置内全部组合。

03

看报告

暖色报告：图表联动、筛选侧栏、航班号/日期/价格明细。

核心能力

  • 往返联票 + 开口程（去程 A→B、返程 C→D，出发/返回均在出发地集合内）
  • 「国庆」→ 10-01~10-07；「国庆前后」→ 9-28~10-10 等规则解析
  • 机场中文搜索 + 城市/机场编码选择，路线优先中文城市名
  • RollingGo 实时查价；HTTP 200 但业务失败时明确提示「查价服务异常」

查价需自备 RollingGo API Key（Web 在设置页配置，Skill 写入 .env）。展示价为成人含税合计参考价，无 OTA 跳转。
