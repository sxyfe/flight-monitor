常见问题

需要注册或付费吗？
不需要。Web 查价与监控工具免费开放，直接在对应页面使用即可。
RollingGo Key 从哪里获取？
前往 rollinggo.store 申请；Key 仅保存在本机设置页，不会上传给开发者。
LLM Key 是必填的吗？
可选。未配置时使用规则回退解析自然语言；配置后可提升复杂意图的解析准确度。
展示的价格能直接订票吗？
否。展示价为 RollingGo 成人含税合计参考价，不含 OTA 跳转，仅供比价决策。
