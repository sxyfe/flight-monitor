首次使用 · 配置 API

第一次使用前完成以下配置；RollingGo Key 为查价必需，LLM Key 仅在自然语言搜索时可选增强解析。

01 必填

申请 RollingGo MCP Key

前往 rollinggo.store 注册并选择 MCP 接入，申请航班 MCP API Key（格式 mcp_...）。此 Key 在本项目各产品中通用。

02 必填

在 API 设置填入并测试

进入 在线查价 · API 设置，粘贴 RollingGo Key → 保存配置 → 点击「测试 RollingGo」（含机场搜索与查价探针）。

03 可选

配置大模型 API Key

仅在自然语言搜索 Tab 使用：将「国庆前后东南亚」等句子解析为查询条件。支持 OpenAI 兼容接口；未配置时自动规则回退。表单查询与价格矩阵不受影响。

04

开始查价

自然语言 / 表单 / 价格矩阵任选其一，选择精简或全量穷举模式，查看暖色报告与图表联动结果。

监控与通知

Flight Watch 同样使用 RollingGo MCP Key（在监控页设置 Tab 配置，无需 LLM）。支持达美开口预设一键导入、筛选排序监控列表，降价时飞书或微信 PushPlus 推送。

  • 监控列表：按启用状态筛选、按价格/名称/最近查价排序，低价命中高亮
  • 预设库：预览航段 → 导入并编辑 → 补全日期后启用

核心能力

  • 往返联票 + 开口程（去程 A→B、返程 C→D，出发/返回均在出发地集合内）
  • 「国庆」→ 10-01~10-07；「国庆前后」→ 9-28~10-10 等规则解析
  • 机场中文搜索 + 城市/机场编码选择，路线优先中文城市名
  • Flight Watch 支持飞书 + 微信 PushPlus 降价通知，达美开口 15 条预设一键导入

RollingGo MCP API Key 在本项目 Web 查价、Flight Watch 与 Skill 中通用（Web 在各自设置页配置，Skill 写入 .env）。LLM Key 仅在线查价的自然语言搜索可选使用。展示价为成人含税合计参考价，无 OTA 跳转。
